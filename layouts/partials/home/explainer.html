{{ if (eq $.Lang "uk" ) }} Чому потрібна модернізація системи спорту? {{ else }} Why sport system modernization is needed? {{ end }}

{{ if (eq $.Lang "uk" ) }}

Спорт є дзеркалом суспільства. В демократичних країнах спорт організований навколо інтересів громадян, які об'єднуються в клуби. В комуністичних країнах спорт є політичним інструментом і організований централізовано навколо інтересів держави. Ці дві моделі управління спортом працюють лише тоді, коли відображають суспільство в якому вони втілені.

Україна успадкувала комуністичну модель від СРСР, проте обрала курс на демократію та ринкову економіку. Механізми та KPI комуністичної системи більше не працюють. Українське суспільство обирає європейський шлях, а отже і Європейську модель спорту.

{{ else }}

Sport is a reflection of the society. Democratic countries have sports system organized around the interests of the citizens, through free association into clubs. In communist countries, sports is a political tool and organized in a centralized fashion around the interests of the state. These two different sport models work only when they match the underlying society.

Ukraine inherited a communist system from USSR as a legacy, but has choosed the path to demoracy and free market. Communist system's methods and KPIs are no longer working. Ukrainian society has choosen European path, and thus need to switch to European sports model.

{{ end }}

{{ if (eq $.Lang "uk" ) }} Що таке коаліція? {{ else }} What is coalition? {{ end }}

{{ if (eq $.Lang "uk" ) }}

Коаліція - це тимчасове об'єднання різних організацій задля вирішення спільної проблеми.

Чим складніша проблема, тим більше ресурсів, досвіду, навичок та зв'язків потрібно для її вирішення. Коаліція дає кожному члену більше ресурсів, зв'язків та можливостей і робить можливими зміни, які не під силу кожному окремо.

Якщо кожен з учасників прагне змін, але не може втілити їх самостійно, то найкращий крок - це об'єднатися у коаліцію. Простими словами: разом можна зробити більше, ніж самотужки.

{{ else }}

Coalition is a temporary alliance of different organizations to solve a common problem.

The more complex the problem, the more resources, experience, skills and connections are needed to solve it. Coalition gives each member more resources, connections and opportunities and makes changes possible that are not possible for each one alone.

If each of the participants wants the change, but cannot implement them alone, then the best step is to unite in a coalition. In simple words: together you can do more than alone.

{{ end }}
