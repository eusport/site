{{ if (eq $.Lang "uk" ) }}

Спорт є дзеркалом суспільства. В демократичних країнах спорт організований навколо інтересів громадян, які об'єднуються в клуби. В комуністичних країнах спорт є політичним інструментом і організований централізовано навколо інтересів держави. Ці дві моделі управління спортом працюють лише тоді, коли відображають суспільство в якому вони втілені.

Україна успадкувала комуністичну модель від СРСР, проте обрала курс на демократію та ринкову економіку. Механізми та KPI комуністичної системи більше не працюють. Українське суспільство обирає європейський шлях і прагне розвитку спортивної галузі, яка пригнічена державними радянськими регулюваннями та спотворенням ринку.

Крім того, комуністична модель суперечить Олімпійській та Європейській спортивній хартії та Конституції України.

{{ if (eq $.Lang "uk" ) }} Що таке Європейська модель спорту? {{ else }} What is the European sport model? {{ end }}

Автономія

Спорт належить суспільству, а не державі.

Держава не втручається в управління спортом, проте виступає надійним партнером спортивної спільноти.

Соціальна функція

Спорт - це перш за все здоров'я та інструмент соціальної інтеграції.

Головний критерій розвитку спорту - це масовість і залученість всіх верств населення до спорту.

Солідарність

Економічна солідарність та волонтерство є запорукою економіки спорту.

Держава є одним з важливих стейкголдерів спорту, але є єдиним і не головним.

{{ if (eq $.Lang "uk" ) }} Що таке коаліція? {{ else }} What is coalition? {{ end }}

{{ if (eq $.Lang "uk" ) }}

Коаліція - це тимчасове об'єднання різних організацій задля вирішення спільної проблеми.

Чим складніша проблема, тим більше ресурсів, досвіду, навичок та зв'язків потрібно для її вирішення. Коаліція дає кожному члену більше ресурсів, зв'язків та можливостей і робить можливими зміни, які не під силу кожному окремо.

Якщо кожен з учасників прагне змін, але не може втілити їх самостійно, то найкращий крок - це об'єднатися у коаліцію. Простими словами: разом можна зробити більше, ніж самотужки.
